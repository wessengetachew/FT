Half-Barrier Depth Theorem

\[ d(r,\,2r+1)\;=\;p_{\min}(r)-2 \]
r = — pmin(r) = — d(r, 2r+1) = — d(r, 2r−1) = 0
Coprimality chain depth at the two fractions nearest to ½ — determined exactly by the smallest prime factor of the numerator.
Theorem Statement ▶
Theorem  (Half-Barrier Depth)

Let \[ d(r,M) \] denote the depth of the coprimality chain, defined as the largest integer $d \ge 0$ such that \[ \gcd(r,\,M+j) = 1 \qquad (0 \le j \le d). \]

Then: \[ d(r,\,2r+1) \;=\; p_{\min}(r) - 2, \] where $p_{\min}(r)$ is the smallest prime divisor of $r$, with the convention $p_{\min}(1)=\infty$, giving infinite depth for $r=1$.

Furthermore, for every $r \ge 2$, \[ d(r,\,2r-1) = 0. \]

Equivalently, among the fractions nearest to $\tfrac{1}{2}$, the fraction \[ \frac{r}{2r+1} \] has depth determined exactly by the smallest prime factor of $r$, while \[ \frac{r}{2r-1} \] fails at the first possible step.

Corollary — Primality Sentinel

$r$ is prime if and only if $d(r, 2r+1) = r - 2$. In particular, the maximum possible depth for numerator $r$ at the upper barrier is exactly $r - 2$, attained precisely when $r$ itself is prime.

Corollary — r = 1 (Infinite Chain)

For $r = 1$: $\gcd(1, M+j) = 1$ for all $j \ge 0$, so $d(1, 3) = \infty$. The fraction $\tfrac{1}{3}$ lies immediately above $\tfrac{1}{2}$ and initiates a chain of infinite depth, consistent with the convention $p_{\min}(1) = \infty$.

Remark — Connection to C(n) and Lift Survival
The depth $d(r, M)$ is exactly the block-coprime depth counted in the Lift Survival Function. The theorem identifies a structural asymmetry at the half-barrier: the upper neighbor $2r+1$ admits a chain whose length is governed by $p_{\min}(r)$, while the lower neighbor $2r-1$ collapses immediately. This asymmetry is reflected in the density function $C(n)$ through the differential contribution of fractions just above and just below $\tfrac{1}{2}$.
Proof
The proof proceeds in two independent parts: the upper barrier $M = 2r+1$ and the lower barrier $M = 2r-1$. Throughout, $r \ge 2$ is a fixed integer with smallest prime factor $p = p_{\min}(r)$. For $r = 1$ the chain is infinite by inspection.
Part I — Upper Barrier: $d(r,\,2r+1) = p_{\min}(r) - 2$
1. Coprimality at $M = 2r+1$ (depth $\ge 0$) ▶

We first verify $\gcd(r,\, 2r+1) = 1$, establishing that the chain begins. Suppose $q \mid r$ and $q \mid 2r+1$. Then $q \mid (2r+1) - 2r = 1$, so $q = 1$. Hence $\gcd(r,\, 2r+1) = 1$ for all $r \ge 1$.

This places $j = 0$ inside the chain: $\gcd(r,\, (2r+1)+0) = \gcd(r,\, 2r+1) = 1$. So $d(r,\, 2r+1) \ge 0$ for all $r \ge 1$.

2. Chain survives for $j = 0, 1, \ldots, p-3$ ▶

Let $p = p_{\min}(r)$. For $0 \le j \le p-3$, we show $\gcd(r,\, 2r+1+j) = 1$.

Since $p \mid r$, we need $p \nmid (2r+1+j)$. We have \[ 2r + 1 + j \;\equiv\; 1 + j \pmod{p}. \] For $0 \le j \le p-3$, the residue $1+j$ ranges over $1, 2, \ldots, p-2$, none of which are $\equiv 0 \pmod{p}$ (since $p \ge 2$ and $1+j \le p-2 < p$).

For any other prime $q \mid r$ with $q > p$: we need $q \nmid (2r+1+j)$. Since $q \mid r$, we have $2r + 1 + j \equiv 1 + j \pmod{q}$. For $j \le p - 3 \le q - 3 < q - 1$, the residue $1 + j$ lies in $\{1, \ldots, q-2\}$, so $q \nmid (2r+1+j)$.

Hence $\gcd(r,\, 2r+1+j) = 1$ for all $0 \le j \le p-3$, giving depth $d \ge p - 2$.

3. Chain fails at $j = p - 2$ (exact termination) ▶

We show $\gcd(r,\, 2r + 1 + (p-2)) > 1$, i.e., the chain fails at $j = p-2$.

Compute: \[ 2r + 1 + (p - 2) \;=\; 2r + p - 1. \] Since $p \mid r$, we have $2r \equiv 0 \pmod{p}$, so \[ 2r + p - 1 \;\equiv\; 0 + 0 - 1 \;\equiv\; -1 \;\equiv\; p - 1 \pmod{p}. \] Wait — this gives residue $p-1 \ne 0$. Let us recount: the chain checks $j = 0, 1, \ldots, d$ where $d$ is the largest such that all checks pass. The check at step $j$ is $\gcd(r,\, M + j) = 1$ with $M = 2r+1$. At $j = p-2$: \[ 2r + 1 + (p-2) = 2r + (p-1). \] Reducing mod $p$: $2r \equiv 0$, so $2r + (p-1) \equiv p - 1 \pmod p$. This is non-zero. We need to find the first $j$ where $p \mid (2r+1+j)$.

We want the smallest $j \ge 0$ with $p \mid 2r + 1 + j$. Since $p \mid r$, $2r \equiv 0 \pmod p$, so $2r + 1 + j \equiv 1 + j \pmod p$. We need $1 + j \equiv 0 \pmod p$, i.e., $j \equiv -1 \equiv p - 1 \pmod p$. The smallest non-negative solution is $j = p - 1$.

So the chain survives for $j = 0, 1, \ldots, p-2$ and fails at $j = p-1$: \[ \gcd(r,\, 2r + 1 + (p-1)) = \gcd(r,\, 2r + p) = p, \] since $p \mid r$ and $p \mid 2r + p$ (as $p \mid 2r$ and $p \mid p$).

The depth — the largest $d$ for which all of $j = 0, \ldots, d$ pass — is thus $d = p - 2$ (the chain passes through $j = p-2$ and fails at $j = p-1$, so the last valid $d$ with $j \le d$ all passing is $d = p - 2$).

Therefore $d(r,\, 2r+1) = p_{\min}(r) - 2$. $\square$

Part II — Lower Barrier: $d(r,\,2r-1) = 0$
4. Immediate failure at $j = 1$ for the lower barrier ▶

First note $\gcd(r,\, 2r-1) = 1$: if $q \mid r$ and $q \mid 2r-1$, then $q \mid (2r - (2r-1)) = 1$. So the chain starts: depth $\ge 0$.

Now examine $j = 1$: \[ \gcd(r,\, (2r-1) + 1) = \gcd(r,\, 2r) = \gcd(r,\, 2r). \] Since $r \mid 2r$, we have $\gcd(r,\, 2r) = r$. For $r \ge 2$ this exceeds 1, so the chain fails at $j = 1$.

Hence the depth is exactly $d(r,\, 2r-1) = 0$ for all $r \ge 2$: the chain begins but fails at the very next step. $\square$

Observation
The lower fraction $r/(2r-1)$ always achieves depth 0 (fails at $j=1$ since $\gcd(r, 2r) = r \ge 2$). The upper fraction $r'/(2r'+1)$ — equivalently $\frac{(M-1)/2}{M}$ for odd $M$ with $r' = (M+1)/2$ — achieves depth $p_{\min}(r') - 2$. When $r'$ is prime, this equals $r' - 2 = (M-1)/2 - 1$, the maximum.
