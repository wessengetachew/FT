Framework — From Grid to Rail

We interpret a G×G grid with coordinates (a,b), 1≤a,b≤G. Each cell represents the fraction a/b. A cell is illuminated (white) iff gcd(a,b)=1 — i.e., the fraction is primitive. Cells with gcd(a,b)>1 represent non-primitive fractions and are dimmed by the sieve.

The sieve depth n controls which primes have been applied. For each prime p≤G, the Euler factor is modified from the classical (1−1/p²) to:

(1 − min(n+1, p) / p²)
n+1 < pPartial binding — factor = 1 − (n+1)/p²
n+1 ≥ pSaturated — factor = 1 − 1/p = (p−1)/p
n = 0Open state — classical ζ(2) coprime density
n ≥ max prime ≤ GFully saturated — ∏(p−1)/p = φ(P)/P

The Rail formula Rail(G,n) = (π²/6)·∏_{p≤G}(1−min(n+1,p)/p²) is always a rational multiple of π². The tables above enumerate every distinct rail value as n sweeps from 0 to full saturation. The key insight: dimming a single pixel and shifting the infinite rail are the same mathematical event.

Special States

For G=3 (primes 2 and 3), three states illustrate the full sieve transition:

n Action Rail Decimal
0 All cells white — open state (1/9)π² ≈ 1.0966
1 Cell (2,2) dims — prime 2 saturated (7/108)π² ≈ 0.6398
2 Cell (3,3) also dims — fully saturated (1/18)π² ≈ 0.5483

The passage from n=0 to n=1 — a single pixel (2,2) going dark — is mathematically identical to multiplying the infinite product by (1−1/4)/(1−1/2) = (3/4)/(1/2) = 3/2 and adjusting, giving the exact jump from π²/9 to 7π²/108.
