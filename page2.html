Geometric Theorem: The Gap Diagonal Identity New · 2026

Consider the integer lattice of pairs $(r, M)$ with $1 \le r, M \le G$. Color a cell gold if $\gcd(r, M) = 1$. Fix any anti-diagonal defined by $|M - r| = g$.

$$\text{density of gold cells on } \{|M - r| = g\} \;=\; \frac{\varphi(g)}{g}$$

This holds for any $g \ge 1$ and any $G \ge g$, uniformly — independent of the grid size. Euler's totient function $\varphi$ is literally the gold density of a diagonal stripe in the coprime lattice.

Proof Sketch

On the diagonal $|M - r| = g$, every cell has the form $(r, r + g)$ for $r \ge 1$. The cell is gold iff $\gcd(r, r+g) = 1$. Since $\gcd(r, r+g) = \gcd(r, g)$, we need $\gcd(r, g) = 1$.

$$\gcd(r,\, r+g) \;=\; \gcd(r,\, g)$$

The proportion of $r \in \{1, \ldots, g\}$ with $\gcd(r,g) = 1$ is exactly $\varphi(g)/g$. The diagonal repeats with period $g$, so every window of $g$ consecutive cells contains exactly $\varphi(g)$ gold ones. The density is therefore $\varphi(g)/g$ for any $G$. $\square$

Step 01Parametrize the diagonal▶

The anti-diagonal $|M - r| = g$ consists of pairs $(r, r+g)$ for $r = 1, 2, \ldots, G-g$ (and symmetrically $(r+g, r)$). We focus on one sheet; the other is identical by symmetry.

Step 02Reduce the coprimality condition▶

A cell $(r, r+g)$ is gold iff $\gcd(r, r+g) = 1$. By the Euclidean algorithm, $\gcd(a, b) = \gcd(a, b-a)$, so $\gcd(r, r+g) = \gcd(r, g)$. The condition simplifies to $r \perp g$.

Step 03Count via periodicity▶

The function $r \mapsto \gcd(r,g)$ has period $g$. In each block of $g$ consecutive integers, exactly $\varphi(g)$ are coprime to $g$. As $r$ runs over the diagonal, this pattern repeats perfectly, giving density $\varphi(g)/g$ independent of $G$.

Step 04Exact for all G ≥ g▶

The diagonal has $G - g$ cells. Since the period-$g$ pattern tiles perfectly, the boundary error is at most $g - 1$ cells, giving observed density within $O(g/G)$ of the exact value $\varphi(g)/g$. In practice, for $G \gg g$ the identity is exact to many decimal places.

Extension to Block-Coprimality — The C(n) SettingExtension

For a fixed depth $n \ge 0$, a cell $(r, M)$ is gold if $\gcd(r, M+j) = 1$ for all $j = 0, 1, \ldots, n$. On the diagonal $|M - r| = g$, this reduces to: $\gcd(r, g+j)=1$ for all $j=0,\ldots,n$.

$$\text{gold density on } |M-r|=g \text{ at depth } n \;=\; \prod_{p \mid g(g+1)\cdots(g+n)} \!\!\left(1 - \frac{1}{p}\right)$$

Gaussian Integer Extension — Coprime Lattice in $\mathbb{Z}[i]$New

Rotating the coprime lattice by $90°$ increments around the origin embeds it into all four quadrants of $\mathbb{Z}[i]$. Each quadrant corresponds to multiplication by a unit $i^k$: the original patch occupies Quadrant IV ($z = r - iM$), and successive $90°$ clockwise rotations fill Q III, Q II, and Q I. Because the Gaussian norm $N(z) = x^2 + y^2$ is invariant, the coprimality structure maps perfectly across all four fields.

The anti-diagonals $|M - r| = g$ (straight lines in the single-quadrant view) become concentric tilted square shells around the origin. The Gap Diagonal Identity becomes a Square Shell Identity: each shell's gold density is exactly $\varphi(g)/g$, by the same periodicity argument, now realized over the entire complex lattice.

3D Coprime Lattice — Joint Coprimality in $\mathbb{Z}^3$

A point $(x, y, z)$ is lit when $\gcd(x, \gcd(y, z)) = 1$ — i.e., no prime divides all three coordinates simultaneously. The density of such triples is $\zeta(3)^{-1} \approx 0.8319$ (Apéry's constant reciprocal).

At $2^n$ grid sizes the outer boundary face locked at $Z = 2^n$ reduces to a strict parity check, producing a perfect 3D checkerboard where every lit block is surrounded on all six sides by dark blocks. The interior carves prime-number tunnels straight through the volume.

$$\text{lit density} = \prod_p \!\left(1 - \frac{1}{p^3}\right) = \frac{1}{\zeta(3)} \approx 0.8319$$
